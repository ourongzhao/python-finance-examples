## 📌 Overview

This repository contains a growing collection of Python notebooks covering key topics in
quantitative finance — built for personal reference, experimentation, and continuous learning.

Each notebook is self-contained, with clear inputs, model implementation, and numerical output.
Topics span interest rate modeling, derivatives valuation, and simulation methods.

---

## 📂 Topics Covered

| Topic | Description |
|---|---|
| 📈 Interest Rate Modeling | Vasicek, CIR, Hull-White, term structure calibration |
| 🔔 Cap & Floor Valuation | Black's model, implied vol, Greeks |
| 🔄 Swaption Valuation | European swaptions, analytical and Monte Carlo methods |
| 🎲 Quasi-Monte Carlo | Sobol and Halton sequences, variance reduction |
| ➕ More coming soon | XVA components, credit risk, portfolio optimization |
